[h1]SchanaModSurvivorSelect[/h1]

An Open Source DayZ mod that provides the ability for players to select their survivor model as a command line parameter when launching the game.

Simply launch the game with the [code]-survivor={name}[/code] parameter with one of the following names to set your character's model. Example: [code]-survivor=Peter[/code]

[b]Male Models:[/b]
Boris, Cyril, Denis, Elias, Francis, Guo, Hassan, Indar, Jose, Kaito, Lewis, Manua, Mirek, Niki, Oliver, Peter, Quinn, Rolf, Seth, and Taiki

[b]Female Models:[/b]
Eva, Frida, Gabi, Helga, Irena, Judy, Keiko, Linda, Maria, and Naomi
